### <span id="使用说明-1">使用说明</span>

* 进入《Dude, Stop》游戏安装目录，并进入 ```Localization``` 文件夹。

* 将本项目的 ```zh_CN``` 文件夹放入该文件夹内。

* 进入游戏，在标题画面中点击 ```OPTIONS``` 选项。

* 点击 ```LANGUAGES``` 切换为中文。
