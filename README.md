### <span id="使用说明-1">使用说明</span>

* 在游戏安装目录下的 ```Localization``` 文件夹下，创建一个新的文件夹并命名为 ```zh_CN```。

* 进入该文件夹，将本项目的所有文件放入该文件夹内。

* 进入游戏，点击```OPTIONS```。

* 点击```LANGUAGES```切换为中文。
